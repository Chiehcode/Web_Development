### Arrow Function

類似 Python 的 Lambda expression

#### JavaScript
```
//JavaScript function
 function funcName(param) {
  return param + 10;
}

//JavaScript arrow function
const funcName = (param) => param + 2
```

#### Python
```
#Python function 
 def func_name(param):
      return param + 10

#Python lambda
func_name = lambda param: param + 10
```
